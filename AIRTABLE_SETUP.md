# Airtable Setup Guide

## Quick Setup Instructions

### Table 1: "Waitlist Signups"

1. **Create the table:**
   - Go to your Airtable base: https://airtable.com/appYT7C2POhuNDsDs
   - Click **"+ Add a table"**
   - Name it exactly: **`Waitlist Signups`** (case-sensitive)

2. **Add these fields** (in this order):
   - **Email** → Field type: **Email** (Required)
   - **Signup Number** → Field type: **Number**
   - **Premium Eligible** → Field type: **Checkbox**
   - **Coupon Code** → Field type: **Single line text**
   - **Date Signed Up** → Field type: **Date** (with time)

### Table 2: "General Inquiries"

1. **Create the table:**
   - Click **"+ Add a table"** (if you don't already have this table)
   - Name it exactly: **`General Inquiries`** (case-sensitive)

2. **Add these fields** (in this order):
   - **Name** → Field type: **Single line text** (Required)
   - **Email** → Field type: **Email** (Required)
   - **Message** → Field type: **Long text** (Required)
   - **Date Submitted** → Field type: **Date** (with time)

## Field Type Reference

When creating fields in Airtable, use these field types:

| Field Name | Airtable Field Type |
|------------|---------------------|
| Email | Email |
| Name, Message, Coupon Code | Single line text (or Long text for Message) |
| Signup Number | Number |
| Premium Eligible | Checkbox |
| Date Signed Up, Date Submitted | Date (enable "Include time") |

## Verification

After creating the tables, you can verify them by:

1. **Running the check script:**
   ```bash
   node scripts/check-airtable.js
   ```

2. **Or manually test:**
   - Try submitting the waitlist form on your website
   - Check if a record appears in the "Waitlist Signups" table
   - Check if an email is sent

## Troubleshooting

### Table not found errors:
- Make sure table names match **exactly** (case-sensitive)
- "Waitlist Signups" not "Waitlist Signup" or "waitlist signups"

### Field errors:
- Field names must match **exactly** (case-sensitive)
- Field types should match the types listed above

### API errors:
- Verify `AIRTABLE_ACCESS_TOKEN` is correct in Cloudflare
- Verify `AIRTABLE_BASE_ID` is correct: `appYT7C2POhuNDsDs`
- Check that your Airtable token has access to this base

## Notes

- Field names are **case-sensitive** - they must match exactly
- Table names are **case-sensitive** - they must match exactly
- The code will automatically populate fields when forms are submitted
- "Signup Number" and "Coupon Code" are auto-generated by the code
- "Premium Eligible" is automatically set based on signup count
